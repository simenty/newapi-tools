## 🔧 修复的关键 Bug

### 1. 递归调用无限循环
**问题**: `init_state()` → `set_state()` → `init_state()` 无限递归  
**修复**: `init_state()` 直接更新时间戳，不再调用 `set_state()`  
**影响**: 状态文件初始化现在可靠

### 2. 嵌套键读取失败
**问题**: 无 jq 时，`get_state("installation.status")` 返回空  
**修复**: 新增 Python 降级方案，支持嵌套键和数组  
**影响**: 无 jq 环境下状态管理正常工作

### 3. Python heredoc 缩进错误
**问题**: Python 代码在 bash heredoc 中缩进错误  
**修复**: 将 Python 代码写入临时文件执行  
**影响**: 无 yq 环境下配置管理正常工作

### 4. 缺失函数导致导出失败
**问题**: `common.sh` 缺少 `log_debug()`，`ui.sh` 缺少 `confirm_action()`  
**修复**: 添加缺失函数  
**影响**: 所有函数正确导出，无未定义错误
